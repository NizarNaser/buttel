# دليل تشغيل نظام butTel - المساعد الصوتي الذكي 🚀

هذا الدليل يشرح لك خطوة بخطوة كيفية تشغيل النظام، المواقع المطلوبة، والإعدادات اللازمة ليعمل التطبيق بكفاءة كاملة.

---

## 1. المتطلبات البرمجية (على جهازك أو الخادم)
قبل البدء، تأكد من تثبيت الأدوات التالية:
- **Node.js** (إصدار 18 فما فوق).
- **MongoDB**: قاعدة بيانات لتخزين معلومات الشركات والسجلات.
- **إضافة Ngrok** (اختياري للبرمجة المحلية): لربط Twilio بجهازك الشخصي.

---

## 2. المواقع التي يجب التسجيل بها (Account Setup)

تحتاج إلى حسابات في المواقع التالية لتفعيل ميزات الذكاء الاصطناعي والاتصال:

### أ- OpenAI (للذكاء الاصطناعي)
- **الموقع**: [platform.openai.com](https://platform.openai.com)
- **الهدف**: الحصول على `OPENAI_API_KEY`.
- **المهمة**: تفعيل نموذج **GPT-4o** الذي يفهم اللغات (العربية، الألمانية، الإنجليزية) ويقوم بالرد الذكي.

### ب- Twilio (للاتصالات الصوتية)
- **الموقع**: [twilio.com](https://www.twilio.com)
- **الهدف**: الحصول على رقم هاتف وتفعيل الاستجابة الصوتية.
- **ماذا تحتاج**:
  - `Account SID`
  - `Auth Token`
  - شراء رقم هاتف (Phone Number) يدعم خاصية الصوت (Voice).

### ج- MongoDB Atlas (اختياري - إذا كنت تريد قاعدة بيانات سحابية)
- **الموقع**: [mongodb.com/cloud/atlas](https://www.mongodb.com/cloud/atlas)
- **الهدف**: الحصول على رابط الاتصال `MONGODB_URI` إذا لم تكن تريد تشغيل MongoDB محلياً.

---

## 3. إعداد ملف الإعدادات (.env)

قم بفتح ملف `.env` في مجلد المشروع ووضع القيم التي حصلت عليها من المواقع أعلاه:

```env
# مفتاح OpenAI
OPENAI_API_KEY=sk-xxxx...

# إعدادات Twilio
TWILIO_ACCOUNT_SID=ACxxxx...
TWILIO_AUTH_TOKEN=xxxx...
TWILIO_PHONE_NUMBER=+123456789

# قاعدة البيانات (محلياً أو سحابياً)
MONGODB_URI=mongodb://localhost:27017/buttel

# حماية لوحة التحكم
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=ضع_هنا_كلمة_سر_عشوائية_طويلة
ADMIN_PASSWORD=admin123
```

---

## 4. خطوات التشغيل (Step-by-Step)

### الخطوة 1: تثبيت المكتبات
افتح التيرمينال داخل مجلد المشروع ونفذ:
```bash
npm install
```

### الخطوة 2: تشغيل التطبيق
للبدء في وضع التطوير:
```bash
npm run dev
```
سيعمل التطبيق الآن على الرابط: `http://localhost:3000`

### الخطوة 3: ربط Twilio بالتطبيق (Webhook)
بما أن Twilio يحتاج للوصول إلى رابط على الإنترنت:
1. إذا كنت تبرمج محلياً، استخدم Ngrok:
   ```bash
   ngrok http 3000
   ```
2. انسخ الرابط الذي سيعطيك إياه Ngrok (مثلاً: `https://abcd-123.ngrok-free.app`).
3. اذهب إلى إعدادات رقم الهاتف في **Twilio Console**.
4. في قسم **Voice & Fax**، وتحت "A CALL COMES IN"، اختر **Webhook** وضع الرابط التالي:
   `https://your-domain.com/api/voice` (استبدل your-domain برابط مشروعك).

## 5. كيفية الاستخدام وتسجيل الشركات (User Guide)

لإعداد النظام للعمل مع شركة جديدة، اتبع الخطوات التالية:

1. **الدخول للوحة التحكم**: اذهب إلى المتصفح وافتح الرابط `http://localhost:3000/admin`.
2. **تسجيل الدخول**: أدخل كلمة مرور الإدارة (الموجودة في ملف `.env` تحت `ADMIN_PASSWORD`).
3. **إضافة شركة جديدة**:
    - اضغط على زر **"+ Add New Company"** الأزرق.
    - **اسم الشركة**: أدخل اسم الشركة كما تريد أن ينطق به الذكاء الاصطناعي.
    - **رقم الهاتف**: أدخل رقم Twilio الذي اشتريته (مثلاً: `+49123456789`).
    - **الخدمات**: اكتب الخدمات مفصولة بفاصلة (مثلاً: محاسبة، مراجعة عقود، ضرائب).
    - **التعليمات الإضافية**: يمكنك كتابة نبرة الصوت المطلوبة أو معلومات سرية لا يعرفها إلا المساعد.
    - اضغط على **"Register Company"** للحفظ.
4. **تفعيل المدفوعات**: من تبويب "Payments"، يمكنك إدخال مفاتيح Stripe أو بريد PayPal لتمكين التحصيل المالي.
5. **استقبال المكالمات**: بمجرد اتصال أي شخص بالرقم المسجل، سيقوم النظام تلقائياً بالتعرف على الشركة والرد بناءً على سياق خدماتها. ستظهر تفاصيل المكالمة والتحليل الذكي فوراً في الصفحة الرئيسية.
